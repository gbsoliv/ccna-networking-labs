# Static Routing and Network Design Lab Series

## Projects

### Part 1 — Static Routing Fundamentals  
Configured communication between two separate LANs using static routing.

### Part 2 — Inter-VLAN Routing (SVI)  
Implemented VLAN segmentation and enabled inter-VLAN communication using Switch Virtual Interfaces.

### Part 3 — Full Network with External Access  
Designed a complete network with multiple VLANs, routing, and connectivity beyond the local network.
