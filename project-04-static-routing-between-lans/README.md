# Project Progression — Networking Labs
This set of projects follows a structured progression from basic routing to a more realistic enterprise network design, building practical experience across Layer 2 and Layer 3.

### Part 1 — Static Routing 
Basic communication between two LANs using static routes. Focus on routing tables, next-hop, and packet flow between networks.

### Project 2 — VLANs & Inter-VLAN Routing (SVI)
Introduced VLAN segmentation and trunking between switches. Configured SVIs on a multilayer switch to enable inter-VLAN communication and act as default gateways.

<img width="1437" height="640" alt="inter-vlan-svi" src="https://github.com/user-attachments/assets/4c2f6c73-9ea2-4610-a3e8-9c6e10836c29" />

### Project 3 — External Connectivity
Extended the internal network by adding a Layer 3 link to a router. Implemented a default route on the switch and return routes on the router to allow traffic to reach external networks.
