
## Overview
This repository contains a collection of hands-on networking labs designed to support my studies for the CCNA 200-301 certification.

The goal is to build a strong foundation in networking concepts while developing practical skills using Cisco Packet Tracer.

---

## What You'll Find Here
Each project in this repository simulates a real-world networking scenario and includes:

- Detailed documentation (README per project)  
- Network topology diagrams  
- Device configurations (CLI)  
- Basic security implementations  
- Testing and validation steps  
- Packet Tracer files (.pkt)
